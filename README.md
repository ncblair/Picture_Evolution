# Picture_Evolution
Classify MNIST_Data, then use an evolutionary algorithm to generate an image resembling the classified number


- Load Input Data

- Preprocess Image Data
  
- Train Simple Net

- Run Evolutionariy Algorithm

- Resize, Denoise and Save Images to File
